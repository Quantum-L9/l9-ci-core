# Roadmap

All four clean-room phases of `l9-ci-core` v2 are implemented:

1. Control-plane foundation and immutable SDK provisioning.
2. Validated artifact protocol and reusable normalization workflows.
3. Execution profiles, rollout modes, requiredness, waivers, and promotion controls.
4. GitHub checks, workflow summaries, bounded SDK-projected annotations, publication metadata, and release authorization.

Provider implementation, provider-native parsing, canonical finding
construction, policy classification, gate computation, AST analysis,
repository graphs, hosted analysis, and canonical bundle merging remain outside
this repository.
